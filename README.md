### Description

这是一个测试的版本，但他确实是可以使用的，用法是将此插件导入到dify工作流中，使用dify生成一段带有html样式（便于匹配代码控制）
同时你需要传入一个模板文档，并调整模板文档和代码中的style对应的内容一致（这么一看用起来还挺不方便）
总之你需要有模板文档，并且有匹配的工作流输出，再加上匹配的style，有了这些你就可以正常使用
